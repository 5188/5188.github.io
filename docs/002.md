#### 模版项目

推荐使用模版项目，因为它为你省去一些项目配置的时间

#### 快速启动

```shell
    
    git clone https://github.com/ezdojs/ezdo-start

    cd ezdo-start && npm install

    npm run start

```

执行上面的几步操作，即可看到一个 ezdo.js 项目已经启动。你可以简单的浏览一下项目代码，我想任何会了解过其他面向对象语言的人都可以很快入手

#### 工具
